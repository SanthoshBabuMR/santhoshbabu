Build and deployment

Create new heroku app

Getting Started with Node.js on Heroku

code block: CLI

// login to heroku
heroku login
<type email and password>
---
// Prepare the app
git clone https://github.com/html5webster/repo-name.git
// navigate to the repo folder
cd repo-name
// Deploy the app
heroku create
---
// push to heroku
git push heroku master

Generate SSH and add to heroku

Adding keys to Heroku

code block: CLI

// open CMD as admin
---
// login to heroku
heroku login
<type email and password>
---
// list avaiable keys
heroku keys
---
// Generate a new SSH key
ssh-keygen -t rsa -C "youremail@abc.com"
---
// Add it to Heroku
heroku keys:add

link local repository to existing heroku app

code block: CLI

git remote add heroku git@heroku.com:name-of-your-heroku-app.git

push BUILD to heroku

code block: CLI

git push heroku master

push CODE to github

code block: CLI

git push origin master
<type email and password>

Undo last git commit

code block: CLI

git reset --hard HEAD~1
