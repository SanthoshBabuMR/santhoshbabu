JS Patterns

Pattern is a reusable software solution to a specific type of problem that occurs frequently when developing software.

Categories Of Design Pattern

  1. Creational Design Patterns

    how to create objects/instances

    Eg: Constructor, Factory, Abstract, Prototype, Singleton and Builder.

  2. Structural Design Patterns

    focuses on object composition i.e. how object's properties ( data and functions ) are structed.

    Eg: Decorator, Facade, Flyweight, Adapter and Proxy.

  3. Behavioral Design Patterns

    how does the objects/instances communicate with each other

    Eg: Iterator, Mediator, Observer and Visitor.
