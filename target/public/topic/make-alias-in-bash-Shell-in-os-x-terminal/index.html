Make alias in bash shall in OS X terminal

  • Launch terminal
  • Go to home directory by typing 
    cd
  • Check if '.bash_profile' file exists in home directory, 
    ls -la
  • If not available, create it using, 
    touch .bash_profile
  • Open file in text edit to add alias, 
    open .bash_profile             // if default editor is text edit
open -a TextEdit .bash_profile // if default editor is other than text edit
  • Add an alias 
    alias documents='cd "/Users/<user_name>/Documents/" && pwd'
  • To add other aliases just start a new line, and apply the same formatting. 
    alias documents='cd "/Users/<user_name>/Documents/" && pwd'
alias downloads='cd "/Users/<user_name>/Downloads/" && pwd'
  • save and close text edit
  • Refresh the bash shell environment: 
    source ~/.bash_profile
