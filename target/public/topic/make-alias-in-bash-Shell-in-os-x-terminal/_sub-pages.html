1. Launch terminal
  2. Go to home directory by typing, 
    cd
  3. Check if '.bash_profile' file exists in home directory, 
    ls -la
  4. If not available, create it using, 
    touch .bash_profile
  5. Open file in text edit to add alias, 
    # default editor is text edit
open .bash_profile 
 
# default editor is other than text edit
open -a TextEdit .bash_profile
  6. Add an alias 
    alias documents='cd "/Users/<user_name>/Documents/" && pwd'
  7. To add other aliases just start a new line, and apply the same formatting. 
    alias documents='cd "/Users/<user_name>/Documents/" && pwd'
alias downloads='cd "/Users/<user_name>/Downloads/" && pwd'
  8. save and close text edit
  9. Refresh the bash shell environment: 
    source ~/.bash_profile
