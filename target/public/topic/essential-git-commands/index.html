Essential GIT commands

Initialize a repo
Create a new local GIT repo git init
Cloning a repo
Clone a repo git clone <remote_repo_url>
Clone a repo with specific folder name git clone <remote_repo_url> <folder_name>
View/Switch/Create/Delete/Merge branch
View branch git branch
View only remote branches git branch -r
View all branches (local & remote) git branch -ra
Switch to an existing branch git checkout <existing_branch_name>
Create a new branch git branch <new_branch_name>
Create and switch to a new branch git checkout -b <new_branch_name>
Delete to a branch git checkout -d <existing_branch_name>
Merge branch git merge <target_branch>
Unable to see a newly created remote branch? # do a 'git pull' to fetch branch updates
git pull
# now do a 'git branch'
git branch
View status, differences and commit changes
View status git status
View difference git diff <file>
Add files to staging git add <file>
git add .
Commit to repo git commit -m "<commit_message>"
Change commit messages
( only if, changes are yet to be pushed to origin ) 		git commit --amend -m "<new_commit_message*gt;"
Undo/Unstage/Delete/Discard
Undo local changes
( local changes will be lost permanently; use with caution ) 		git checkout -- <file-or-folder>
Unstage file git reset <file-or-folder>
Unstage all files git reset
Delete a file from repo and working directory git rm <file-or-folder>
Delete a file only from repo git rm --cached <file-or-folder>
Discard all local changes and unstage all files
( local changes will be lost permanently; use with caution ) 		git reset --hard
Discard last commit; Discard all local changes and unstage all files
( 'git reset' will erase 'history'; use with caution ) 		git reset --hard HEAD~1
Pull and push changes
Pull changes from repo git pull
Pull changes from different branch git pull origin <branch>
Push local changes to repo git push
Push local *branch* to repo git push -u origin <local_branch_name>
Force push local changes to repo git push -f
Create/View/Apply/Drop Stash
Create stash git stash save "<save_message>"
View stash git stash list
Apply stash git stash apply stash@{<index>}
Drop stash git stash drop stash@{<index>}
View commits
View last 5 commit history git log -5
View file stats (added/deleted/modified) git log --stat
View the full diff of each commit. git log -p
View last 5 commit history in condensed format git log -5 --oneline
View last 5 commit containing the specific file git log -5 --oneline -- src/public/main.css
View last 5 commit by specific author git log -5 --oneline --author="<name>"
View last 5 commit by specific message pattern git log -5 --oneline --grep="<pattern>"
Configure user name and email
Configure user name globally git config --global user.name "<name>"
Configure user name in the current repo git config user.name "<name>"
Configure user email globally git config --global user.email "<email>"
Configure user email in the current repo git config user.email "<email>"
View/Edit git configuration file
View configuration file git config --system --list
git config --global --list
git config --local --list
Open the configuration file for manual editing git config --system --edit
git config --global --edit
git config --local --edit
View/Add remote
View remote git remote -v
Add remote origin git remote add origin <remote_repo_url>
