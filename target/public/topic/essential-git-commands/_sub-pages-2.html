Create a new GIT repo

git init

Example #01:

git init

This creates '.git' subdirectory which contains the Git repo's metadata. This command can be used to create a new local Git repo or convert an existing, unversioned project a Git repository

Add file(s) to staging area

git add <path/to/file>

Example #01:

git add src/public/script/main.js
git add src/public/view/about-us.html src/public/view/contact-us.html

git add .

Example #02:

git add .

This will add modified files in working directory to staging area

git add <path/to/directory>

Example #02:

git add src/public/script/vendor/

This will add all modified files in 'vendor' directory to staging area
