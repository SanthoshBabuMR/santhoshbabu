JSON data containing quotes(single/double) breaks when stored in DOM ( through form input's value attribute or data-* attribute )

The data had to be stringified and serialized before being stored in DOM;
Likewise, the data had to be de-serialized and parsed when read from DOM and used in js; 

// writing data to DOM
var saveData = { "comment": "This data contains single (') quote and double (\") quote ;" }
$('body').data('field-data', window.encodeURIComponent(JSON.stringify(saveData)));
// reading data to DOM
var retrieveData = JSON.parse(window.decodeURIComponent($('body').data('field-data')));

Setting default value

I used to follow the below two approaches, to set a default value, if one is not provided

code block: Using ternary operator

function setData (name, rating) {
  var person = name   ? name   : '<some-default-value>';
  var rating = rating ? rating : '<some-default-value>';
  return {
    person: person,
    rating: rating
  };
}
// setData();
// setData('Banana', 6);
// setData('Snacks', 0);
// setData('', 4);

code block: Using OR operator

function setData (name, rating) {
  var person = name   || '<some-default-value>';
  var rating = rating || '<some-default-value>';
  return {
    person: person,
    rating: rating
  };
}
// setData();
// setData('Banana', 6);
// setData('Snacks', 0);
// setData('', 4);

However, with javascript the following are considered falsy:

  • 0 (zero)
  • "" (empty string)
  • null.
  • undefined.
  • NaN (a special Number value meaning Not-a-Number!)

Which implies. if the actual value provided is 'zero' (0) or an empty string (''), the default value will be set instead of the actual value.

A better approach would be to check if the variable in question has any value; This can be achieved by using typeof operator (or) by checking the variable against 'undefined' value.

code block: Using typeof operator

function setData (name, rating) {
  var person = typeof name   !== "undefined" ? name   : '<some-default-value>';
  var rating = typeof rating !== "undefined" ? rating : '<some-default-value>';
  return {
    person: person,
    rating: rating
  };
}
// setData();
// setData('Banana', 6);
// setData('Snacks', 0);
// setData('', 4);

code block: Checking against 'undefined'

function setData (name, rating) {
  var person = name   !== undefined ? name   : '<some-default-value>';
  var rating = rating !== undefined ? rating : '<some-default-value>';
  return {
    person: person,
    rating: rating
  };
}
// setData();
// setData('Banana', 6);
// setData('Snacks', 0);
// setData('', 4);
